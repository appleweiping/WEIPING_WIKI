# 立党 决策记录与行动分析

## 调研时间：2026-05-18

## 重大决策

### 1. 创建 Hedgehog Lab / Hedgehog Computing
- **背景**：在Texas创业，做浏览器端科学计算
- **逻辑**：将科学计算从桌面端搬到浏览器，降低使用门槛
- **结果**：2.4k stars，有一定社区影响力，但非商业大成功
- **决策框架**：技术可行性 + 降低门槛 = 值得做

### 2. 创建 DARC 项目
- **背景**：Web3/区块链热潮期间
- **逻辑**：将公司治理搬到链上，用Plugin-as-a-Law替代传统DAO
- **结果**：9.2k stars，技术上有创新性
- **决策框架**：看到DAO的缺陷（无规则），提出"有规则的链上公司"

### 3. 写「How-to-run」转码指南
- **背景**：自己转码成功后，系统化分享路径
- **逻辑**：信息不对称是痛点，系统化路径指南有价值
- **结果**：5.8k stars，成为华人转码社区标杆资源
- **决策框架**：自己走过的路 + 系统化 = 可复制的价值

### 4. 从"润学推广"到"劝退润学"
- **背景**：2022-2023年润学热潮
- **逻辑**：早期认为润是好选择，后来发现很多人不适合/算不清账
- **结果**：Substack文章「我为什么劝退润学」「不要润，要留在中国」
- **决策框架**：根据新信息修正立场，不执着于过去的判断
- **言行一致性**：显示了愿意公开修正立场的特质

### 5. 持续推荐Georgia Tech OMSCS（10年）
- **背景**：$7000-9000的在线CS硕士
- **逻辑**：性价比最高、Top校名、适合在职/转码人群
- **结果**：成为他最持久的推荐之一
- **决策框架**：成本/收益比 + 长期验证 = 持续推荐

### 6. 投资策略：买SP500/纳100
- **背景**：写在X profile标签里
- **逻辑**：指数投资是普通人最优策略（推测）
- **结果**：作为个人品牌标签持续使用
- **决策框架**：简单 + 长期有效 + 不需要专业知识 = 最优默认选择

## 决策框架总结

立党的决策模式呈现以下特征：

1. **成本敏感**：所有推荐都强调具体成本（$7000、100万RMB）
2. **路径可复制性**：只推荐自己验证过或有大量成功案例的路径
3. **愿意修正**：从润学到劝退润学，显示不执着于过去判断
4. **极端化表达但理性决策**：标题极端，但实际建议通常有数据支撑
5. **长期主义验证**：10年推荐同一个项目，说明重视长期验证

## 言行一致性分析

- **一致**：转码路径建议与自己的实际经历一致
- **一致**：创业指南与自己在Delaware注册公司的经历一致
- **变化**：润学立场有明显转变，但他公开承认了这个变化
- **未验证**：投资建议（SP500/纳100）的个人执行情况未知

## 信息可信度

- GitHub项目决策：一手，高可信度
- 立场变化：一手（Substack文章），高可信度
- 决策框架推断：基于公开行为推断，中等可信度
