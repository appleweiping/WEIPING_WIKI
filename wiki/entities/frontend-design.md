## Usage

Ask for a frontend artifact and name the skill when desired:

```text
用 frontend-design 做一个有强烈记忆点的产品首页。
Use frontend-design to redesign this dashboard UI.
Build a production-grade component with a bold editorial aesthetic.
```

The expected process:

1. Identify purpose, audience, tone, constraints, and the memorable design idea.
2. Choose a deliberate aesthetic direction.
3. Implement working code.
4. Verify the result visually when a runnable frontend exists.
