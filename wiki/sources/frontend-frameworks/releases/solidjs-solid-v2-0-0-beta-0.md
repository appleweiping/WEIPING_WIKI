## Summary

I know you all probably weren't expecting this announcement next. But after reviewing the roadmap, we spent so long iterating in the Experimental phase, most of the goalposts within Alpha don't appear relevant enough to warrant their own phase. Instead, I'm happy to announce today that Solid 2.0 effort enters the beta stage. There will be bugs and rough edges we need to work through with the ecosystem.
