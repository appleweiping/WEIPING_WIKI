# 2026-05-17 LLM Hex Anime Video Project

## Provenance

- Source: user request to implement a WeChat Channels-first short video about mainstream LLM ability hexagons, personified as Japanese anime characters.
- Project path: `D:/video creation/projects/llm-hex-anime/`.

## Outputs Created

- EXTRACTED: `data/models.json` defines ten models: GPT-5.5, Opus, Sonnet, Haiku, DeepSeek Pro, DeepSeek Flash, Gemini, Qwen, Kimi, and Doubao.
- EXTRACTED: Six axes are reasoning depth, coding, writing, multimodal/vision, speed, and cost/accessibility.
- EXTRACTED: `storyboard/shots.json` defines a 16-shot, 71-second vertical-video storyboard.
- EXTRACTED: Prompt packs were created for character cards and video-model shot generation.
- EXTRACTED: Local scripts generate placeholder character cards, stable radar charts, a cover image, SRT subtitles, publishing metadata, and a preview MP4.
- EXTRACTED: Preview video output: `D:/video creation/projects/llm-hex-anime/final/videohao-1080x1920.mp4`.
- EXTRACTED: 视频号 upload command template: `D:/video creation/projects/llm-hex-anime/publish/videohao-command.ps1.txt`.

## Boundaries
