## Summary

Twoolpy Twitter API fun scripts using Tweepy in Python. I'm hoping people can use these examples as repository to draw on when they develop their own scripts. Scripts 1. $ python recommend.py blahblah : gives recommendations to user 'blahblah' on who to follow, based on their social graph on twitter. It looks for people who are very often followed by blahblah's followers, but who 'blahblah' doesn't follow. 2. $ python userstream.py : prints authenticated user's timeline, live 3. $ python followersdiff.py : finds all your followers and saves them to local file.
