## Summary

build nanoGPT This repo holds the from-scratch reproduction of $1. The git commits were specifically kept step by step and clean so that one can easily walk through the git commit history to see it built slowly. Additionally, there is an accompanying $1 where you can see me introduce each commit and explain the pieces along the way. We basically start from an empty file and work our way to a reproduction of the $1 (124M) model. If you have more patience or money, the code can also reproduce the $1 models.
